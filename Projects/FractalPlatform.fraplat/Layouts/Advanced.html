How It Works

Straightforward process, powerful results

Let platform generate most of your website automatically. Customize everything what you need:

  • Create documentation which describes user roles, business flows, user screens and other requirements - your technical specification level
  • Create your data model in document oriented collection. Convert documents to relation database if needed - your database level
  • Fractal automatically creates CRUD business logic - your business logic level
  • Customize your business logic to your purposes - your business logic level
  • Fractal automatically creates UI around data model and provide possibility to customize UI - your user interface level
  • Create integrations with other systems with REST api if needed - your integration level
  • Customize your external UI by AI or Figma where is important unique design. Keep existing auto-generated design for internal screens used by internal personal roles - your user interface level
  • Create dev/uat/prod environments to build full CI/CD processes across team in your own cloud: AWS or Azure. - your CI/CD level
  • Set up automated testing scenarios to validate every new version of your application before deployment. - your auto testing level
  • Keep open door to extend your project on more complex scenarios

How to Get Started

Create, debug and deploy your unique web site on your own

Follow these simple steps to get started:

  • Download Visual Studio .NET or VS Code
  • Clone https://github.com/fraplat/FractalPlatform git hub repository
  • Review existing example projects how it was created
  • Create your own project of your application
  • Set BaseUrl and DeploymentKey in appsettings.json of FractalPlatform.Deployment project
  • Run FractalPlatform.Deployment project to deploy your own web application to the cloud
