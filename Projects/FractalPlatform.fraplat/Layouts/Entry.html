How It Works

From data model to working application in minutes

Just create a JSON of your data model and Fractal Platform:

  • Creates documented or relational database around JSON (your database level)
  • Automatically creates CRUD business logic (your business logic level)
  • Automatically creates UI around data model and provides possibility to customize UI (user interface level)
  • Keeps the door open to extend your project for more complex scenarios

  "Todo": {

  }
