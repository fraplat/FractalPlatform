How It Works

Straightforward process, powerful results

Build your complex project from scratch—designed to adapt and scale as your requirements evolve:

  • Create documentation which describes architecture, user roles, business flows, user screens and other requirements of the system - your technical specification level
  • Dedicate set of applications that should communicate to each other like micro services - your microservice level
  • Creates document or relational databases - your database level
  • Create your data model in document oriented collection. Convert documents to relation database if needed - your database level
  • Fractal automatically creates CRUD business logic - your business logic level
  • Customize your business logic to your purposes - your business logic level
  • Fractal automatically creates UI around data model and provide possibility to customize UI if needed - your user interface level
  • Create integrations with other systems with REST api if needed - your integration level
  • Customize your external UI by AI or Figma where is important unique design. Keep existing auto-generated design for internal screens used by internal personal roles - your user interface level
  • Create dev/uat/prod environments to build full CI/CD processes across team in your own cloud: AWS or Azure. - your CI/CD level
  • Set up automated testing scenarios to validate every new version of your application before deployment - your auto testing level
  • Tune storage system to work with large amount of data - your performance tuning level
  • Add cache levels to each storage to speed up solution - your performance tuning level
  • Add local offline application databases that can be synchronized to master - your distributed architecture level
